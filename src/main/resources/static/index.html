Browsing and modifying JPA entities

Browse and modify entities

In case you started this app from the src/test/java/**/TestDemoApplciation, like you should have, you can though usethe entity explorer view (configured to be available during testing) to browse the db (and to make tiny changes).

Note, that the entity explorere uses Vaadin Flow. In case you plan to use Vaadin, be sure to change dependencies and configs from the test side to the main scope/sources!.

Alternatively you can find out your Docker container running the Sakila DB and open a direct DB connection to the server.
